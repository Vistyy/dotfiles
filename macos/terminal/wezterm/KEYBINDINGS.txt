Terminal cheatsheet (WezTerm + tmux)

WezTerm (macOS):
  Cmd+Shift+h     Help (this screen)
  Cmd+Shift+r     Reload WezTerm config (shows toast)
  Cmd+Shift+t     New WezTerm tab (uses current cwd)
  Ctrl+Cmd+f      Toggle fullscreen

tmux-first shortcuts (via WezTerm; persistent across reattach):
  Cmd+d           tmux split (left/right)
  Cmd+Shift+d     tmux split (top/bottom)
  Cmd+Alt+i/j/k/l Move between tmux panes (up/left/down/right)
  Cmd+Alt+Arrows  Move between tmux panes
  Cmd+t           tmux new window
  Cmd+[           tmux previous window
  Cmd+]           tmux next window
  Cmd+w           tmux kill pane (no confirm) / else close WezTerm tab (confirm)
  Cmd+Shift+w     tmux kill window (confirm) / else close WezTerm tab (confirm)

tmux essentials (works in any client: SSH/phone/WezTerm):
  Prefix          Ctrl+b
  Windows         Prefix+c (new), Prefix+n/p (next/prev), Prefix+w (list), Prefix+1..9 (jump)
  Panes           Prefix+% (L/R split), Prefix+" (T/B split), Prefix+o (next pane), Prefix+arrows (move)
  Kill            Prefix+x (kill pane, no confirm), Prefix+& (kill window, confirm)
  Detach          Prefix+d
  Panes (no prefix) Alt+i/j/k/l (up/left/down/right)

WezTerm leader (Ctrl+a):
  Ctrl+a then r          Reload config
  Ctrl+a then [          Copy mode (scrollback selection)
  Ctrl+a then /          Search scrollback
  Ctrl+a then o          Quick-open URL from screen
  Ctrl+a then w          Launcher (workspaces/tabs/domains)
  Ctrl+a then ,          Rename tab

Help:
  Ctrl+a then Shift+h    Help (this screen)

Reload (alternate):
  Cmd+Alt+r              Reload WezTerm config (shows toast)

Debug:
  Cmd+Alt+D              Show debug overlay

Multiline input:
  Shift+Enter            Insert newline without running the command

Word movement / deletion:
  Opt+Left/Right         Move by word
  Opt+Backspace          Delete previous word
  Opt+d                  Delete next word

Tips:
  Use tmux for session persistence (reattach after closing the app).
