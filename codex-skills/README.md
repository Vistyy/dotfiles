# codex-skills

Codex personal skills vendored into this repo.

## Layout

Each skill lives in its own folder:

```
codex-skills/<skill-name>/SKILL.md
```

## Install

Run `./install.sh` to sync each `codex-skills/<skill-name>` directory into `~/.codex/skills/<skill-name>`.

## Editing

Treat the copy in this repo as the source of truth and re-run `./install.sh` after changes.
